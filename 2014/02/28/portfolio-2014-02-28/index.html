投資組合@2014-02-28

發表於 2014-02-28 | 分類於 投資組合

美股走勢十分凌厲，個人認為NASDAQ今年挑戰2000年科網泡沫高位的機會很大。投資組合得益新能源相關股份大升而水漲船高，但不要太亢奮，經驗告訴筆者，手風太順時，投資組合隨時會地震，做好風險管理為上策。

本周新資金注入投資組合，按之前部署，先減持較弱的太陽能股SPWR，然後分散到其他牛板塊，如Biotech和科網。

現在組合的思路清楚，Big Cap核心有MPEL、LVS、REGN和TSLA，太陽能最有信心是SCTY，其次是CSIQ,SUNE,SPWR，PLUG和YY則較有賭博成份。

操作：
減持SPWR
增持REGN
買入YY
