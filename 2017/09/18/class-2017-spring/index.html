我的教育之路：2017春

發表於 2017-09-18 | 分類於 創業日誌

這個學期的課是同一間中學，同樣是兩班(一班高中，一班初中)，但人數增至每班7-8學生，主要增加了一些新來港學生。問題來了，我們從來沒試過這個師生比例，可能你會說，也不過是一對單位數的師生比例，有什麼難度？但如果說，每個人學的東西不同，進度不同，作為老師你怎樣管理好這個班，而不讓教學質素下降？這個情況1對1已不容易，更何況1對3、1對4。我們不得不重新設計整個教學模式，並且優化 Neu Academy 學習平台，令這個教學模式得以實行。

我們理想中的模式，是1老師和1助教對每班15學生或以上。要做到這個師生比例，基本上就不可以全程緊貼學生去指導了。於是，我們就設計出一套新的教學模式 (我們叫它 Neu Classroom)，學生一組組輪流進行「老師活動」，其他學生透過電腦在 Neu Academy 學習平台進行「自學活動」。

「老師活動」類似之前1對1的老師指導，但我們嘗試把英文程度相若的學生歸入同一組，以便進行1對2甚至1對3的指導。當然，之前我們要做一些英文測試，評定學生的英文程度。

至於「自學活動」，我們要保證教材要足夠多元化，做到總有一款符合學生的口味(興趣和程度)。於是，我們便建立起教材庫，大量製作不同口味的教材。另外，我們進一步遊戲化 Neu Academy 學習平台，加入虛擬奬勵。其中有學生為拿到所有星星(最高為5星)，重複練習，堪稱完美主義者。「自學活動」自動化，我們可以抽空幫助有需要的學生，從電腦紀錄，老師更容易看到學生遇到的障礙，從而作出糾正，例如我們課後聽學生的錄音，幫他們改善發音的問題。

課程中我們亦遇到一些難題。一，在第一課，學生需要在 Neu Academy 學習平台建立帳號，這花了不少時間，而且忘記帳號和密碼的情況時有發生。解決方法，我們決定在下一次在課前先為所有學生建立帳號，學生直接登入便可，他們有時間可以自行更改。二，麥克風經常出現問題，我估計是平常太少用麥克風，機器少用自然會壞掉，你懂的。解決方法，學校換了一批新的耳機和麥克風。三，其中一課租用不了電腦室，沒有電腦，我們的教學模式便實行不了，這課我們唯有改為全人手，做一些不太個人化的分組活動。解決方法，其實學校有一個多媒體室，可以用iPad上課，可惜那時我們的手機APP還沒開發完成，用不了多媒體室，於是，我們便加快開發，終於在暑期班用得上手機APP。

下回預告，暑期班，15人班的挑戰⋯⋯
